## v2 모델 개선사항

### 1. Attention 안정화 (attention.py)

```python
# kappa 증가 속도 10배 감소
kappa = k_prev + F.softplus(kappa) * 0.1

# beta 범위 제한 (attention이 너무 퍼지는 것 방지)
beta = F.softplus(beta).clamp(min=0.1, max=10.0)
```

**효과**: Attention이 텍스트를 천천히, 정확하게 따라감

### 2. LSTM 장기 의존성 개선 (lstm.py)

```python
# Forget gate bias를 1로 초기화
nn.init.constant_(self.W_f.bias, 1.0)
```

**효과**: 초기 학습 시 정보를 더 오래 유지 (LSTM 표준 트릭, Jozefowicz et al. 2015)

### 3. 과적합 방지 (synthesis.py)

```python
self.dropout = nn.Dropout(0.1)
# MDN 레이어 직전에 적용
h_combined = self.dropout(h_combined)
```

**효과**: 일반화 성능 향상

### 4. 학습 안정화 (trainer.py)

```python
# Learning rate warmup (5 epochs)
self.warmup_epochs = 5
if self.current_epoch < self.warmup_epochs:
    warmup_lr = self.base_lr * (self.current_epoch + 1) / self.warmup_epochs

# Gradient clipping 강화
self.grad_clip = 5  # 기존 10에서 감소
```

**효과**: 초기 학습 안정화, gradient 폭발 방지

### 5. 이전 개선사항 (v1.5에서 적용, 유지)

**mdn.py**:
```python
# sigma 최소값 보장
sigma_adjusted = sigma * max(temp, 0.5)

# eos 선명화
eos_threshold = eos ** (1.0 / max(temp, 0.5))
```

**losses.py**:
```python
# eos 가중치 2배
loss = stroke_nll + 2.0 * eos_bce
```

## v2 모델 + 단어 데이터 학습

v2 모델 문장 데이터로 1 epoch만 진행 후, 단어 데이터로 전환.

```bash
caffeinate -dims python scripts/train.py \
    --config configs/default.yaml \
    --train_data ./data/train_words.h5 \
    --val_data ./data/val_words.h5 \
    --epochs 300 \
    --early_stop_patience 50
```

### 기대 효과

1. 짧은 텍스트(이름/서명)에 특화된 학습
2. 에포크당 학습 시간 단축 (스트로크 길이 1/5)
3. 서명 생성 품질 향상
